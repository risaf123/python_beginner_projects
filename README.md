# Python Beginner Projects

This repository contains a collection of Python projects for beginners. Each project is designed to help you learn core Python concepts and practice problem-solving skills.  

## Projects List

1. **COMPOUND INTEREST CALCULATOR.py**  
   Calculate compound interest for given principal, rate, and time.

2. **alarm_clock.py**  
   Set an alarm and play a sound at the specified time.

3. **armstrong.py**  
   Check whether a number is an Armstrong number.

4. **dail pad.py**  
   Create a simple daily notepad or task list.

5. **encrypt_decrypt.py**  
   Encrypt and decrypt messages using basic cipher techniques.

6. **fire_alarm.mp3**  
   Sound file used by the alarm clock programs.

7. **hangman.py**  
   Play a text-based Hangman game.

8. **prime_no.py**  
   Check whether a number is prime.

9. **quiz game.py**  
   Multiple-choice quiz game.

10. **shopping cart program.py**  
    Simulate a shopping cart with product selection and total calculation.

11. **slot_gamepy.py**  
    Simple slot machine game.

12. **theatre shoppping.py**  
    Simulate a theatre ticket booking system.

13. **timer clock.py**  
    Countdown timer program.

---

## Features

- Covers basic Python concepts like loops, conditions, functions, and input/output.  
- Interactive programs for practice.  
- Simple GUI/audio integration in alarm and timer projects.  
- Fun games to make learning Python enjoyable.
